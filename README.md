# Darkness-Detector

The Darkness Detector is an electronic project that automatically detects the absence of light and switches on a device such as an LED or bulb. It mainly uses an LDR (Light Dependent Resistor) to sense light intensity. When there is sufficient light, the resistance of the LDR is low and the circuit remains OFF. When it becomes dark, the resistance increases, activating a transistor or relay that turns ON the light.
This project is useful for automatic street lights, garden lights, and energy-saving systems. It helps in reducing power consumption and operates without manual control.
